## Getting Started

1. **Install Dependencies**:
   ```bash
   bun install
   ```

2. **Configure Environment**:
   Create a `.env.local` file with your Cosmic credentials:
   ```env
   COSMIC_BUCKET_SLUG=your-bucket-slug
   COSMIC_READ_KEY=your-read-key
   COSMIC_WRITE_KEY=your-write-key
   ```

3. **Run Development Server**:
   ```bash
   bun run dev
   ```

4. **Open Browser**:
   Navigate to `http://localhost:3000` to see the app.

## Deployment

### Vercel
1. Push your code to a Git repository.
2. Import the project into Vercel.
3. Add the Environment Variables (`COSMIC_BUCKET_SLUG`, `COSMIC_READ_KEY`, `COSMIC_WRITE_KEY`) in the Vercel dashboard.
4. Deploy!

### Netlify
1. Connect your repository to Netlify.
2. Configure build settings (Command: `npm run build`, Directory: `.next`).
3. Add Environment Variables in "Site settings" > "Build & deploy" > "Environment".
4. Deploy site.
